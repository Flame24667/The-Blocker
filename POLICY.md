Privacy Policy for The Blocker

The Blocker is a browser extension designed to block ads and trackers and show users what was blocked while browsing.

Data We Handle

The extension may process website URLs, domains, network request URLs, blocked request information, and local extension settings in order to detect, block, and display ads and trackers.

This information is used only to provide the extension’s core features: blocking ads and trackers, showing blocking status, displaying blocked request counts, and saving user preferences.

Local Storage

The Blocker stores settings and blocking statistics locally in the user’s browser storage. This allows the extension to remember preferences and show blocked ad/tracker information.

Data Sharing

The Blocker does not sell user data. The Blocker does not transfer user data to third parties for advertising, profiling, creditworthiness, lending, or unrelated purposes.

Remote Code

The Blocker does not use remote executable code. Extension code is included inside the extension package.

Limited Use Disclosure

The use of information received from Chrome extension permissions will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
